## 這是**TVS資料檔讀取**程式：
* **可使用快捷鍵選擇開啟和幫助**
* **可顯示你要讀的英文單字資料**<br><br>

## **功能介紹**：
1. **開啟檔案**：會顯示四個類別分別的內容。<br>
<img width="777" height="606" alt="image" src="https://github.com/user-attachments/assets/c6a93df0-9f31-4f1f-9388-fe8988ede761" /><br>
2. **左下角訊息**：會顯示成功截入幾個單字。
3. **縮放大小**：可自由調整畫面大小，最小長寬有限制。
4. **幫助的關於選項**：會顯示about表單相關訊息。<br>
<img width="787" height="607" alt="image" src="https://github.com/user-attachments/assets/b835d84d-27a9-46b1-a8f1-d306b433a58c" /><br>
5. **離開**：會做二次確認是否要離開。
